Brotbackuhr ist eine Art 'Eieruhr', damit man den Backofen nicht vergisst.
Einstellbare Zeiten für das  An- und Ausbacken in einem Backvorgang.
(z.B.: 15 Minuten bei 250 °C, anschließend 45 Minuten bei 190 °C)
